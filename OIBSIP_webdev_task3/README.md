## Project Title
Full Stack Pizza Ordering Application

## Objective
To design and develop a complete full-stack web application that allows users to customize pizzas, place orders, and simulate online payments, while providing an admin inventory management system.

## Description
This project is developed as part of **Level 3** of the OIB-SIP Web Development Internship.  
It demonstrates real-world full-stack development using frontend, backend, database integration, and payment workflow concepts.

## Project Structure
- client: Frontend application for users
- server: Backend APIs and business logic

## Key Features
- User registration and login
- Pizza customization (base, sauce, cheese, veggies)
- Order placement and confirmation
- Admin inventory management
- Payment integration (test mode)
- Secure authentication and authorization

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- Razorpay (Test Mode)

## Outcome
Successfully built and tested a real-world full-stack application, gaining practical experience in frontend-backend integration, authentication, database operations, and payment workflows.
